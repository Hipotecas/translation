## 安装并生成Prisma客户端

要开始使用Prisma Client，你需要安装`@prisma/client`包:

```terminal copy
npm install @prisma/client
```

注意，[`@prisma/client`节点模块](../../../../concepts/components/prisma-client/working-with-prismaclient/generating-prisma-client#the-prismaclient-npm-package)引用了一个名为`.prisma/client`的文件夹。`.prisma/client`文件夹包含你生成的prisma客户端，并且每次你改变模式并运行以下命令时都会被修改:

```terminal copy
npx prisma generate
```

这个命令读取你的Prisma模式并生成你的Prisma Client库:

![安装并生成Prisma Client](https://res.cloudinary.com/prismaio/image/upload/v1628761155/docs/FensWfo.png)

`@prisma/client`节点模块引用一个名为`.prisma/client`的文件夹，其中包含您唯一的生成的prisma客户端：

![.prisma和@prisma文件夹](./prisma-client-node-module.png)
