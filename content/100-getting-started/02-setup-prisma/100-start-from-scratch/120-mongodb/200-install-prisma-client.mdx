## 安装并生成Prisma Client

要开始使用 Prisma Client，您需要安装`@prisma/client`软件包：

```terminal copy
npm install @prisma/client
```

注意，安装命令会自动`prisma generate`为您调用，它会读取您的 Prisma 模型并生成一个为您的模型量身定制的 Prisma 客户端版本。

![安装并生成 Prisma Client](https://res.cloudinary.com/prismaio/image/upload/v1628761155/docs/FensWfo.png)

将来，无论何时更改Prisma模式，都需要手动调用`prisma generate`，以适应Prisma客户端API中的更改。
