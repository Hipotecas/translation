## 连接你的数据库

要连接您的数据库，您需要将Prisma 架构`url`中的`datasource`块字段设置为您的数据库连接 [连接URL](../../../../../reference/database-reference/connection-urls)：

```prisma file=prisma/schema.prisma
datasource db {
  provider = "mongodb"
  url      = env("DATABASE_URL")
}
```

在本例中，`url`是[通过环境变量设置](../../../ guides/development-environment/environment-variables)，定义在`.env`(示例使用[MongoDB Atlas](https://www.mongodb.com/cloud/atlas) URL):

```bash file=.env
DATABASE_URL="mongodb+srv://test:test@cluster0.ns1yp.mongodb.net/myFirstDatabase"
```

因为MongoDB目前是一个预览功能，你需要在[`generator`](https://www.prisma.io/docs/getting-started/setup-prisma/start-from-scratch/mongodb)块中明确定义它。

```prisma file=prisma/schema.prisma
generator client {
  provider        = "prisma-client-js"
  previewFeatures = ["mongoDb"]
}
```

现在需要调整连接URL以指向您自己的数据库。

数据库连接 URL 的格式取决于您使用的数据库。对于 MongoDB，它看起来如下（拼写为全部大写的部分是您的特定连接详细信息的 _占位符_）：

```no-lines
mongodb://USERNAME:PASSWORD@HOST:PORT/DATABASE
```

以下是每个组件的简短说明：

- `USERNAME`: 你的数据库用户名
- `PASSWORD`: 数据库用户的密码
- `HOST`: 运行`mongod`](https://docs.mongodb.com/manual/reference/program/mongod/#mongodb-binary-bin.mongod)或[`mongos`](https://docs.mongodb.com/manual/reference/program/mongos/#mongodb-binary-bin.mongos）实例的主机
- `PORT`: 数据库服务器运行的端口(通常为MongoDB的`27017`)
- `DATABASE`: 数据库名称

例如，对于托管在[MongoDB Atlas](https://www.mongodb.com/cloud/atlas)上的 MongoDB 数据库，[连接URL](../../../../reference/database-reference/connection-urls)可能类似于：

```bash file=.env
DATABASE_URL="mongodb+srv://test:test@cluster0.ns1yp.mongodb.net/myFirstDatabase"
```
