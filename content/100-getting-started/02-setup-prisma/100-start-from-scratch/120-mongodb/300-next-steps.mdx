## 下一步

本节列出了您现在可以从这里采取的一些潜在的后续步骤。随意探索这些内容或阅读[简介](../../../../concepts/overview/what-is-prisma)页面以获取对 Prisma 的高级概述。

### 继续探索 Prisma Client API

您可以使用 Prisma Client API 发送各种查询。查看[API 参考](../../../../concepts/components/prisma-client)并使用本指南中现有的数据库设置来试用它们。

<Tip>

您可以使用编辑器的自动完成特性来了解不同的API调用及其参数。自动完成通常通过按键盘上的<kbd>CTRL</kbd>+<kbd>SPACE</kbd>来调用。

</Tip>

<details><summary>展开以获得更多Prisma Client API示例</summary>

对于您可以使用 Prisma Client 发送的更多查询，以下是一些建议：

**过滤所有`Post`包含的记录`"hello"`**

```js
const filteredPosts = await prisma.post.findMany({
  where: {
    OR: [{ title: { contains: 'hello' } }, { body: { contains: 'hello' } }],
  },
})
```

**创建一个新的`Post`记录，并将其连接到一个现有的`User`记录**

```js
const post = await prisma.post.create({
  data: {
    title: 'Join us for Prisma Day 2020',
    slug: 'prisma-day-2020',
    body: 'A conference on modern application development and databases.',
    user: {
      connect: { email: 'hello@prisma.com' },
    },
  },
})
```

**使用fluent relations API通过遍历关系来检索`User`的`Post`记录**

```js
const user = await prisma.comment
  .findUnique({
    where: { id: '60ff4e9500acc65700ebf470' },
  })
  .post()
  .user()
```

**删除一条`User`记录**

```js
const deletedUser = await prisma.user.delete({
  where: { email: 'sarah@prisma.io' },
})
```

</details>

### 在Prisma Studio中探索数据

Prisma Studio是数据库中数据的可视化编辑器。
你可以通过两种方式运行它:

1. 在您的终端中运行 `$ npx prisma studio`。
2. 安装[桌面应用](https://github.com/prisma/studio/releases)程序。支持 Windows、macOS 和 Linux。
