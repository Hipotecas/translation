## <inlinecode>标题</inlinecode>

页面的`<h1>` 标题。

## <inlinecode>网页标题</inlinecode>

页面的 `<title>` - 调为 `title` (`h1`)

## <inlinecode>导航标题</inlinecode>

允许你为左侧导航指定不同的、通常较短的标题。

## <inlinecode>元描述</inlinecode>

页面的 `<meta name="description" content="" />` 。

## <inlinecode>静态链接</inlinecode>

在左侧导航中将页面转换为未链接的标题。例如: https://www.prisma.io/docs/concepts/overview

> **注意**: 该页面仍然存在，但你只能通过面包屑导航到该页面。

## <inlinecode>预览</inlinecode>

在左侧导航的页面中添加`preview`标签。

## <inlinecode>目录</inlinecode>

开启或关闭页面目录导航(默认为`false`)。例如：:

```
toc: true
```

## <inlinecode>目录层级</inlinecode>

控制标题在目录中显示的层级：

```
tocDepth: 2
```

> **注意**: 目前仅限`h2`和`h3`两个级别

## <inlinecode>隐藏页面</inlinecode>

对所有导航隐藏页面(通过搜索仍可见)。例如:

```
hidePage: true
```
