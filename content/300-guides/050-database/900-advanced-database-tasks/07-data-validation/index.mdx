所有的数据库都包括一个验证输入的机制。在 SQL 环境中，`CHECK`约束允许指定一组必须满足的条件 - 例如，列 X 不应该大于列 Y。本节描述了如何配置自定义`CHECK`约束。

已经可以在 Prisma schema 中定义以下约束：

- [Field must be unique](../../../../concepts/components/prisma-schema/data-model#defining-a-unique-field) <span class="concept"></span> (`UNIQUE` constraint)
- [Field is mandatory](../../../../concepts/components/prisma-schema/data-model#optional-and-mandatory-fields) <span class="concept"></span> (`NOT NULL` constraint)

</TopBlock>
