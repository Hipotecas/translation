<TopBlock>

Prisma 当前支持以下数据库:

| 数据库                       | 版本 |
| ---------------------------- | ---- |
| PostgreSQL                   | 9    |
| PostgreSQL                   | 10   |
| PostgreSQL                   | 11   |
| PostgreSQL                   | 12   |
| PostgreSQL                   | 13   |
| MySQL                        | 5.6  |
| MySQL                        | 5.7  |
| MySQL                        | 8    |
| MariaDB                      | 10   |
| SQLite                       | \*   |
| AWS Aurora                   | \*   |
| AWS Aurora Serverless &sup1; | \*   |

请注意，每个 Prisma 版本都附带了固定版本的 SQLite。

&sup1; 不支持[Data API for Aurora Serverless](https://github.com/prisma/prisma/issues/1964).

</TopBlock>
